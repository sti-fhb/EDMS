# User Story 6 — UCET009 線上測驗作答（ET06）

> 返回總檔：[spec.md](spec.md) | 模組：教育訓練文件管理（ET）

學員於 ET05 章節學習中遇到測驗項目時，進入 ET06 線上測驗作答頁。流程依序為：引導頁（顯示測驗名稱、題數、及格分數、作答時間限制、剩餘重考次數、上次成績）→ 答題介面（題號進度 / 倒數計時 / 提交按鈕、左側題目導覽列可任意跳題、單選 radio / 多選 checkbox、切換題目自動暫存）→ 提交後即時自動閱卷顯示總分與答題明細（強制顯示正確答案）。多選題採部分計分公式 `max(0, (對 − 誤) ÷ 應選 × 配分)`。題目順序與選項順序於開始作答時系統自動洗牌並寫入 `ET_QUIZ_ATTEMPT_M` 快照（同一 attempt 內順序固定；重考時重新洗牌）。未及格且有剩餘重考次數時可立即重考（無冷卻）。學員可回看自己**每一次** attempt 之成績與逐題明細（不限最近一次；依各 attempt 之順序快照渲染）。課程關閉當下若有作答中 attempt，沿用快照**允許完成並計分**；之後不可再開新作答。

**Priority**: P1

**Why this priority**: 學員核心考核作業；學員之測驗成績為完課判定與平均成績計算之依據。

**Independent Test**: 學員完成一次測驗 attempt（單選 + 多選各至少一題），驗證自動閱卷分數與答題明細正確；多選題部分計分公式套用無誤；未及格時可立即重考且題目重新洗牌。

**Acceptance Scenarios**:

### 引導頁

1. **Given** 學員於 ET05 點擊章節中之測驗項目，**When** 系統載入，**Then** 顯示引導頁：測驗名稱、題數、及格分數、作答時間限制、剩餘重考次數、上次成績（若有）
2. **Given** 測驗之作答時間限制 = 0，**When** 學員於章節學習頁，**Then** 該測驗項目**直接隱藏不顯示**（視同無測驗、不阻擋下一章節解鎖）
3. **Given** 學員已用完重考次數且未及格，**When** 進入引導頁，**Then** 「開始測驗」按鈕禁用；提示「重考次數已用完，請聯繫教師重置」

### 答題介面

4. **Given** 學員點「開始測驗」，**When** 系統建立 ET_QUIZ_ATTEMPT_M，**Then** 寫入「題目順序快照（QUESTION_ORDER）+ 各題選項順序快照（OPTION_ORDER）+ 題目 / 選項 / 配分 / 及格分數 / 作答時間限制版本快照」
5. **Given** 學員進入答題介面，**When** 系統載入，**Then** 上方狀態列顯示題號進度、倒數計時、提交按鈕；左側題目導覽列依快照順序呈現
6. **Given** 學員作答單選題，**When** 點 radio 選項，**Then** 系統暫存當前作答
7. **Given** 學員作答多選題，**When** 勾選多個 checkbox，**Then** 系統暫存
8. **Given** 學員切換至其他題號，**When** 系統暫存當前題答案後載入下一題，**Then** 順序依快照保持一致
9. **Given** 學員作答中跳題、暫存後回來，**When** 系統載入，**Then** 題目順序與選項順序**保持一致**（同 attempt 內固定）

### 計時與提交

10. **Given** 學員作答中，倒數計時歸零，**When** 系統偵測，**Then** 以當前作答狀態自動提交
11. **Given** 學員作答中嘗試切換瀏覽器頁面 / 關閉視窗，**When** 系統偵測 unload 事件，**Then** 提示確認避免誤離（瀏覽器 native confirm）
12. **Given** 學員點「提交」，**When** 系統收到，**Then** 即時自動閱卷並顯示總分、是否及格、答題明細

### 自動閱卷與部分計分

13. **Given** 單選題答對，**When** 閱卷，**Then** 該題得分 = 配分；答錯則為 0 分
14. **Given** 多選題（應選 = 3、配分 = 30），學員答對 2 個、誤選 1 個，**When** 閱卷，**Then** 該題得分 = max(0, (2 − 1) ÷ 3 × 30) = 10
15. **Given** 多選題學員選擇之錯選數 > 應選數，**When** 閱卷，**Then** 套用公式後得 0 分（公式之 max 保護）
16. **Given** 多選題學員完全未勾選任何選項，**When** 閱卷，**Then** 該題視為 0 分
17. **Given** 測驗總分 ≥ 及格分數，**When** 系統判定，**Then** is_pass = true；該章節解鎖
18. **Given** 測驗總分 < 及格分數，**When** 系統判定，**Then** is_pass = false；該章節未解鎖；若剩餘重考次數 > 0 顯示「重新作答」按鈕

### 答題明細與重考

19. **Given** 學員提交完成，**When** 載入答題明細頁，**Then** 顯示每題之：題型、學員當次選擇、正確答案、結果（答對 / 部分對 / 答錯）、得分（**強制顯示正確答案**，無教師可關閉選項）
20. **Given** 學員未及格、有剩餘重考次數，**When** 點「重新作答」，**Then** 立即建立新 ET_QUIZ_ATTEMPT_M 並重新洗牌題目順序與選項順序（無 cooldown）
21. **Given** 學員重考多次，**When** 計算結業成績，**Then** **以該學員之最高分為基礎**（per RQ「以最高分為結業成績」）

### 歷次作答明細回看（2026-07-02 新增）

22. **Given** 學員曾於某測驗作答多次，**When** 學員自引導頁點「歷次作答紀錄」，**Then** 系統列出**每一次** attempt（作答時間、總分、是否及格），不限最近一次
23. **Given** 學員點選歷次清單中之某次 attempt，**When** 系統載入，**Then** 依**該次 attempt 之題目 / 選項順序快照**渲染逐題明細（題目、學員當次作答、正確答案、對錯、該題得分）
24. **Given** 教師端需檢視同一資料，**When** 教師於 ET03 點擊學員，**Then** 可展開該學員之歷次 attempt 清單與逐題明細（屬 [spec_us9.md](spec_us9.md) US9）
25. **Given** 課程已關閉，**When** 學員以唯讀回看模式進入，**Then** 仍可回看自己歷次 attempt 之成績與逐題明細（唯讀；不可開新作答）

### 並發處理與課程關閉

26. **Given** 學員作答中，教師修改測驗內容（題目 / 選項 / 配分），**When** 學員提交，**Then** 沿用 attempt 開始時之快照（不影響本次 attempt）；新開啟之 attempt 才套用最新版
27. **Given** 學員作答中，課程被**關閉**（到期自動或教師手動，屬 [spec_us11.md](spec_us11.md)），**When** 系統處理，**Then** 課程**立即轉 CLOSED**（狀態不延遲）；作答中 attempt **沿用快照允許完成並計分**（含 timeout 自動提交），提交後正常閱卷並計入歷史紀錄；之後**不可再開始新的作答**（含重考）；學員端其餘功能即時轉唯讀（2026-07-02 變更：原 PENDING_CLOSE 過渡狀態移除）
28. **Given** 學員作答中，教師將該學員從課程移除，**When** 學員提交，**Then** 當前 attempt 完成並計入歷史紀錄；學員 next navigation 被導至「您已被該課程移除」訊息頁

## Functional Requirements

- **FR-ET-US6-01**: 系統 MUST 於引導頁顯示測驗名稱、題數、及格分數、作答時間限制、剩餘重考次數與上次成績（若有）；學員已用完重考次數且未及格時 MUST 禁用「開始測驗」按鈕並提示聯繫教師重置。
- **FR-ET-US6-02**: 系統 MUST 於測驗之作答時間限制 = 0 時將該測驗項目隱藏不顯示（視同無測驗），且 MUST NOT 以該測驗阻擋下一章節解鎖。
- **FR-ET-US6-03**: 系統 MUST 於學員開始作答時建立 ET_QUIZ_ATTEMPT_M 並寫入題目順序（QUESTION_ORDER）、各題選項順序（OPTION_ORDER）及題目 / 選項 / 配分 / 及格分數 / 作答時間限制之版本快照；同一 attempt 內順序 MUST 固定，重考時 MUST 重新洗牌。
- **FR-ET-US6-04**: 系統 MUST 於答題介面顯示題號進度、倒數計時與提交按鈕，提供左側題目導覽列任意跳題，並於學員切換題目時自動暫存當前作答；題目與選項順序 MUST 依 attempt 快照保持一致。
- **FR-ET-US6-05**: 系統 MUST 於倒數計時歸零時以當前作答狀態自動提交，並於學員作答中嘗試切換頁面 / 關閉視窗時提示確認以避免誤離。
- **FR-ET-US6-06**: 系統 MUST 於學員提交後即時自動閱卷並顯示總分、是否及格與答題明細。
- **FR-ET-US6-07**: 系統 MUST 以「答對得配分、答錯得 0 分」計算單選題，以部分計分公式 `max(0, (對 − 誤) ÷ 應選 × 配分)` 計算多選題（負值由公式之 max 保護為 0、完全未作答視為 0 分）。
- **FR-ET-US6-08**: 系統 MUST 於總分 ≥ 及格分數時判定 is_pass = true 並解鎖該章節；於總分 < 及格分數時判定 is_pass = false、不解鎖該章節，且於剩餘重考次數 > 0 時提供「重新作答」。
- **FR-ET-US6-09**: 系統 MUST 於答題明細強制顯示每題之題型、學員當次選擇、正確答案、結果（答對 / 部分對 / 答錯）與得分，MUST NOT 提供教師關閉正確答案顯示之選項。
- **FR-ET-US6-10**: 系統 MUST 於學員點「重新作答」時立即建立新 ET_QUIZ_ATTEMPT_M 並重新洗牌題目與選項順序（無 cooldown）；計算結業成績時 MUST 以該學員之最高分為基礎。
- **FR-ET-US6-11**: 系統 MUST 提供學員回看自己每一次 attempt 之成績與逐題明細（不限最近一次），並依各該次 attempt 之題目 / 選項順序快照渲染。
- **FR-ET-US6-12**: 系統 MUST 於學員作答中教師修改測驗內容時沿用該 attempt 開始時之快照（僅新開啟之 attempt 套用最新版）；課程關閉時 MUST 立即轉 CLOSED，允許作答中 attempt 沿用快照完成並計分（含 timeout 自動提交）後計入歷史紀錄，MUST NOT 於關閉後開始新的作答（含重考）；學員作答中遭移除者，當前 attempt 完成並計入歷史後，於 next navigation 導至「您已被該課程移除」訊息頁。

## 相關 Clarifications 摘錄

- 多選題部分計分公式：`max(0, (對 − 誤) ÷ 應選 × 配分)`；題目建立時強制至少 1 正確選項；完全未作答視為 0 分
- 題目與選項順序於 attempt 開始時凍結並寫入 ET_QUIZ_ATTEMPT_M 快照
- 重考無 cooldown；以最高分為結業成績；強制顯示正確答案（無教師設定關閉之選項）
- 學員可回看自己**每一次** attempt 之成績與逐題明細（不限最近一次；依各 attempt 之快照渲染）；教師端同等資料於 US9 檢視（2026-07-02 新增）
- 課程關閉當下之作答中 attempt 沿用快照允許完成並計分；之後不可開新作答；原 PENDING_CLOSE 過渡狀態移除（2026-07-02 變更，關閉已可逆、無需終態保護）
