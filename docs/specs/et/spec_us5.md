# User Story 5 — UCET008 章節學習（ET05）

> 返回總檔：[spec.md](spec.md) | 模組：教育訓練文件管理（ET）

學員於 ET04 點擊課程卡片進入 ET05 章節學習頁。系統載入該課程之章節導覽列（左側），中間內容區顯示當前章節之教材（影片 / DM 文件 / 說明文字）。學員依章節順序學習，章節解鎖規則為：含影片之章節需所有影片**累計覆蓋率 ≥ 80%**（允許跳轉、重複觀看不加成）；僅含文件 / 說明文字之章節，學員開啟即視為已閱讀（純記錄不強制）；含測驗之章節須測驗達及格分數方可解鎖下一章節（測驗作答屬 [spec_us6.md](spec_us6.md) US6）。學員觀看位置跨 session 保留，返回時自動恢復。

**Priority**: P1

**Why this priority**: 學員核心學習作業，課程內容之主要呈現面；強制完成解鎖規則為訓練成效之保證機制。

**Independent Test**: 學員以瀏覽器播放影片至 80% 覆蓋率，驗證下一章節解鎖；故意快轉至最末，驗證覆蓋率未達 80% 時下一章節仍鎖定；返回課程後播放位置正確恢復。

**Acceptance Scenarios**:

### 載入與導覽

1. **Given** 學員於 ET04 點擊課程卡片，**When** 系統載入 ET05，**Then** 左側顯示章節導覽列（含章節 / 教材 / 測驗清單，標示已完成 / 進行中 / 未解鎖狀態）；中間為內容區
2. **Given** 學員首次進入該課程，**When** 系統載入，**Then** 自動定位至第 1 章節之第 1 項目
3. **Given** 學員已學習過該課程，**When** 系統載入，**Then** 自動定位至上次觀看位置（依 ET_PROGRESS 之 LAST_POSITION）

### 影片播放

4. **Given** 學員開啟影片教材，**When** 系統載入，**Then** 以 HTML5 video player 呈現（含播放 / 暫停 / 調速 / 音量 / 全螢幕控制）
5. **Given** 學員播放影片並暫停 / 跳轉 / 結束，**When** 系統偵測動作，**Then** 對應之播放區段以 INSERT 寫入 ET_PROGRESS_INTERVAL
6. **Given** 學員離開頁面（瀏覽器 unload 事件），**When** 系統收到，**Then** 對該影片之 ET_PROGRESS_INTERVAL 執行 normalize：SELECT 全部 → 排序 → 合併重疊 / 鄰近區段 → DELETE 全部 → INSERT 合併後結果
7. **Given** 學員瀏覽器強制關閉 / 斷網未正常離開，**When** 學員下次開啟並離開該影片，**Then** 系統補做 normalize；覆蓋率計算前後皆正確

### 解鎖規則

8. **Given** 學員於含影片之章節觀看，累計覆蓋率達 80%，**When** 系統判定，**Then** 下一章節解鎖
9. **Given** 學員於含影片之章節觀看，累計覆蓋率僅 70%，**When** 學員嘗試點擊下一章節，**Then** 系統阻擋並提示「請先完成本章節之影片學習」
10. **Given** 學員於含測驗之章節，已通過教材條件但測驗未及格，**When** 學員嘗試點擊下一章節，**Then** 系統阻擋並提示「請通過本章節之測驗後解鎖」
11. **Given** 章節僅含文件 / 說明文字，**When** 學員開啟該項目，**Then** 系統標記為已閱讀；下一章節無條件解鎖
12. **Given** 學員觀看影片但反覆 seek 同一段，**When** 系統計算覆蓋率，**Then** 重複觀看區段聯集去重，不重複計算

### DM 文件呈現

13. **Given** 學員開啟 DM 文件教材，**When** 文件為 PDF，**Then** 於頁內嵌入預覽
14. **Given** 學員開啟 DM 文件教材，**When** 文件為非 PDF（如 Excel、Word），**Then** 顯示「下載原檔以本機應用程式開啟」連結
15. **Given** 學員開啟之 DM 文件已被廢止，**When** 系統載入，**Then** 顯示「此文件已廢止」標籤；學員仍可閱讀廢止前最後版本

### 並發處理

16. **Given** 學員作答 / 觀看中，教師端刪除該章節，**When** 學員下次切換頁面 / 重新整理，**Then** 顯示「此內容已刪除」訊息
17. **Given** 課程被停課（屬 [spec_us11.md](spec_us11.md)），**When** 學員於 ET05 重新整理或切換章節，**Then** 顯示「此課程已停止開放」訊息頁

---

## 相關 Clarifications 摘錄

- 影片 80% 採累計覆蓋率（區段聯集去重）；允許跳轉、重複觀看不加成
- ET_PROGRESS_INTERVAL 不限區段筆數；學員離開頁面時 normalize
- 教材內容可重複觀看；跨章節保持上次觀看位置
