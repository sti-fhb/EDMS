# User Story 5 — UCET008 章節學習（ET05）

> 返回總檔：[spec.md](spec.md) | 模組：教育訓練文件管理（ET）

學員於 ET04 點擊課程卡片進入 ET05 章節學習頁。系統載入該課程之章節導覽列（左側），中間內容區顯示當前章節之教材（影片 / DM 文件 / 說明文字）。學員依章節順序學習，章節解鎖規則為：含影片之章節需所有影片**累計覆蓋率 ≥ 80%**（實際播放區段聯集 ÷ 影片總長；允許拉進度條跳轉但**僅實際播放區段計入、直接拉到結尾不算看過**；允許倍速播放且**倍速照算覆蓋率**，倍速選項 0.75 / 1 / 1.25 / 1.5 / 2、上限 2 倍；重複觀看不加成）；僅含文件 / 說明文字之章節，學員開啟即視為已閱讀（純記錄不強制）；含測驗之章節須測驗達及格分數方可解鎖下一章節（測驗作答屬 [spec_us6.md](spec_us6.md) US6）。學員觀看位置跨 session 保留，返回時自動恢復。學員**完課後**（課程有課後問卷時）章節導覽列顯示「**填寫課後問卷**」入口，導向 US13（spec_us13.md）填寫。課程**已關閉**時內容轉**唯讀回看**（可重看已學過的教材），不再累積進度、不可作答、不可解鎖新章節、不可填問卷。

**Priority**: P1

**Why this priority**: 學員核心學習作業，課程內容之主要呈現面；強制完成解鎖規則為訓練成效之保證機制。

**Independent Test**: 學員以瀏覽器播放影片至 80% 覆蓋率，驗證下一章節解鎖；故意將進度條直接拉到結尾，驗證未實際播放之區段不計入覆蓋率、下一章節仍鎖定；以 2 倍速實際看完全片，驗證覆蓋率照算為 100%；返回課程後播放位置正確恢復。

**Acceptance Scenarios**:

### 載入與導覽

1. **Given** 學員於 ET04 點擊課程卡片，**When** 系統載入 ET05，**Then** 左側顯示章節導覽列（含章節 / 教材 / 測驗清單，標示已完成 / 進行中 / 未解鎖狀態）；中間為內容區
2. **Given** 學員首次進入該課程，**When** 系統載入，**Then** 自動定位至第 1 章節之第 1 項目
3. **Given** 學員已學習過該課程，**When** 系統載入，**Then** 自動定位至上次觀看位置（依 ET_PROGRESS 之 LAST_POSITION）

### 影片播放

4. **Given** 學員開啟影片教材，**When** 系統載入，**Then** 以 HTML5 video player 呈現（含播放 / 暫停 / 調速 / 音量 / 全螢幕控制）；倍速選項為 **0.75 / 1 / 1.25 / 1.5 / 2**（上限 2 倍，由 `ET_PARAM.VIDEO_PLAYBACK_MAX_RATE` 控制）
5. **Given** 學員播放影片並暫停 / 跳轉 / 結束，**When** 系統偵測動作，**Then** 對應之播放區段以 INSERT 寫入 ET_PROGRESS_INTERVAL
6. **Given** 學員離開頁面（瀏覽器 unload 事件），**When** 系統收到，**Then** 對該影片之 ET_PROGRESS_INTERVAL 執行 normalize：SELECT 全部 → 排序 → 合併重疊 / 鄰近區段 → DELETE 全部 → INSERT 合併後結果
7. **Given** 學員瀏覽器強制關閉 / 斷網未正常離開，**When** 學員下次開啟並離開該影片，**Then** 系統補做 normalize；覆蓋率計算前後皆正確

### 解鎖規則

8. **Given** 學員於含影片之章節觀看，累計覆蓋率（實際播放區段聯集 ÷ 影片總長）達 80%，**When** 系統判定，**Then** 下一章節解鎖
9. **Given** 學員於含影片之章節觀看，累計覆蓋率僅 70%，**When** 學員嘗試點擊下一章節，**Then** 系統阻擋並提示「請先完成本章節之影片學習」
10. **Given** 學員將進度條**直接拉到結尾**（未實際播放中間內容），**When** 系統計算覆蓋率，**Then** 跳過之區段**不產生觀看區段、不計入覆蓋率**（僅實際播放過的區段計入；允許跳轉但不算看過）
11. **Given** 學員以 **2 倍速**實際播放完全片，**When** 系統計算覆蓋率，**Then** 覆蓋率 = 100%（**倍速播放之區段照算覆蓋率**，不採牆鐘時間計算）
12. **Given** 學員於含測驗之章節，已通過教材條件但測驗未及格，**When** 學員嘗試點擊下一章節，**Then** 系統阻擋並提示「請通過本章節之測驗後解鎖」
13. **Given** 章節僅含文件 / 說明文字，**When** 學員開啟該項目，**Then** 系統標記為已閱讀；下一章節無條件解鎖
14. **Given** 學員觀看影片但反覆 seek 同一段，**When** 系統計算覆蓋率，**Then** 重複觀看區段聯集去重，不重複計算（重複觀看不加成）

### DM 文件呈現

15. **Given** 學員開啟 DM 文件教材，**When** 文件為 PDF，**Then** 於頁內嵌入預覽
16. **Given** 學員開啟 DM 文件教材，**When** 文件為非 PDF（如 Excel、Word），**Then** 顯示「下載原檔以本機應用程式開啟」連結
17. **Given** 學員開啟之 DM 文件已被廢止，**When** 系統載入，**Then** 顯示「此文件已廢止」標籤；學員仍可閱讀廢止前最後版本

### 課後問卷入口（2026-07-02 新增）

18. **Given** 學員已完課且課程**設有課後問卷**，**When** 系統載入 ET05，**Then** 章節導覽列顯示「**填寫課後問卷**」入口；點擊導向課後問卷填寫（屬 US13，spec_us13.md）
19. **Given** 課程**未設**課後問卷，**When** 系統載入 ET05，**Then** 「填寫課後問卷」入口**不顯示**
20. **Given** 學員尚未完課，**When** 系統載入 ET05，**Then** 「填寫課後問卷」入口不顯示（完課後方開放填寫）
21. **Given** 學員已填過問卷，**When** 點擊該入口，**Then** 顯示自己已填之內容（唯讀；一人一次，送出後不可修改）

### 並發處理與課程關閉

22. **Given** 學員作答 / 觀看中，教師端刪除該章節，**When** 學員下次切換頁面 / 重新整理，**Then** 顯示「此內容已刪除」訊息
23. **Given** 課程被關閉（到期自動或教師手動，屬 [spec_us11.md](spec_us11.md)），**When** 學員於 ET05 重新整理或切換章節，**Then** 內容轉**唯讀回看**模式：可重看已學過的教材，但**不再累積進度**（不寫入 ET_PROGRESS / ET_PROGRESS_INTERVAL）、不可作答測驗、不可解鎖新章節、不可填問卷；頁面顯示「此課程目前關閉中」提示標示（非阻擋進入之訊息頁）
24. **Given** 課程已關閉後教師執行「再開課」（重設起訖時間，屬 [spec_us11.md](spec_us11.md)），**When** 學員再次進入 ET05，**Then** 恢復正常學習模式；學員進度接續保留

## 系統訊息

各訊息類型（錯誤 / 警告 / 確認 / 成功 / 提示）定義見 [spec.md](spec.md) §Requirements。ET05 為多 US 共用畫面，本 US（章節學習）使用流水號區段 001–099。

| 訊息代碼 | 類型 | 訊息內容 | 觸發情境 |
|---------|------|---------|---------|
| ET-MSG-ET05-001 | 警告 | 請先完成本章節之影片學習 | 場景 9：影片累計覆蓋率未達 80% 嘗試解鎖下一章節 |
| ET-MSG-ET05-002 | 警告 | 請通過本章節之測驗後解鎖 | 場景 12：測驗未及格嘗試解鎖 |
| ET-MSG-ET05-003 | 警告 | 此文件已廢止 | 場景 17：DM 文件已廢止（仍可閱讀廢止前最後版本）|
| ET-MSG-ET05-004 | 提示 | 此內容已刪除 | 場景 22：教師刪除學員正在檢視之章節 |
| ET-MSG-ET05-005 | 提示 | 此課程目前關閉中，僅可回看已學內容 | 場景 23：課程關閉轉唯讀回看 |

## 相關 Clarifications 摘錄

- 影片 80% 採累計覆蓋率（實際播放區段聯集 ÷ 影片總長；區段聯集去重）；允許拉進度條跳轉但僅實際播放區段計入、**直接拉到結尾不算看過**；重複觀看不加成；不採「結束時間 − 開始時間」之牆鐘時間計算（2026-07-02 明文化）
- 倍速播放照算覆蓋率（2 倍速實際看完全片 = 覆蓋率 100%）；倍速選項 0.75 / 1 / 1.25 / 1.5 / 2，上限 2 倍由 `ET_PARAM.VIDEO_PLAYBACK_MAX_RATE` 控制（2026-07-02 明文化）
- ET_PROGRESS_INTERVAL 不限區段筆數；學員離開頁面時 normalize
- 教材內容可重複觀看；跨章節保持上次觀看位置
- 完課後（課程有問卷時）章節導覽列顯示「填寫課後問卷」入口（2026-07-02 新增）；填寫流程屬 US13；填寫問卷**不是**完課條件、不計入學習進度 %
- 課程關閉＝**唯讀回看**（可見、可回看已學內容），非「停止開放」擋門；關閉可逆，教師可再開課（2026-07-02 變更）
