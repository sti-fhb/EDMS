# User Story 9 — UCET005 學員學習狀況追蹤（ET03）

> 返回總檔：[spec.md](spec.md) | 模組：教育訓練文件管理（ET）
> 2026-07-02 變更：ET03 改為「已加入」「待加入」**兩個 tab**；「問卷結果」由獨立 tab **併入「已加入」頁籤**。「已加入」頁籤為一個課程之完整資料視圖，分**三區塊**：①已加入學員 ②作答明細 ③問卷結果。

教師於 ET03 學員頁透過下拉選單切換要檢視之課程；「已加入」「待加入」兩個 tab 同步以該課程過濾。「**已加入**」頁籤為該課程之完整資料視圖，分三區塊呈現：

- **區塊 1 — 已加入學員**：該課程之已加入學員清單，欄位包含學員姓名、加入日期、完課狀態（已完成 / 進行中 / 未開始）、學習進度、平均成績、最後活動時間。操作僅有**移除學員**（ET_ENROLLMENT 標記移除、前台不再顯示、學習歷史保留、不計入完課率統計）。（作答明細與重置重考次數一律於區塊 2 進行，區塊 1 操作欄不另設入口）
- **區塊 2 — 作答明細**（2026-07-02 新增）：**一次列出所有曾作答之學員**（可展開清單，不設學員篩選）；展開某學員即見其**歷次（每一次）attempt 清單**（作答時間、總分、是否及格），點入單次 attempt 檢視逐題明細。各測驗標題列設**重置重考次數**按鈕（**以該測驗為單位**；限該學員於該測驗之重考次數已用盡且未及格時可點，已及格 / 未作答則停用）
- **區塊 3 — 問卷結果**（2026-07-02 新增；課程無課後問卷時隱藏）：**統計檢視**（各題選項分布人數與百分比、已填 / 未填人數）與**明細檢視**（逐學員具名之各題選擇），可匯出 CSV

清單支援匯出 CSV（依當前篩選條件）。課程**已關閉**時教師仍可閱覽三區塊全部內容，但不可重置重考次數 / 移除學員 / 邀請。「待加入」tab 屬 [spec_us12.md](spec_us12.md) US12。

**Priority**: P2

**Why this priority**: 教師追蹤學員學習成效之主要工具；雖非學員端必要作業，但教師需要此頁判斷課程效果與重考必要性。

**Independent Test**: 對於有多名學員之課程，於 ET03「已加入」頁籤區塊 1 可見所有學員之完課狀態與平均成績；區塊 2 列出所有曾作答學員、展開某學員可見其歷次 attempt 清單並點入單次逐題明細；有問卷之課程於區塊 3 可查得各題選項分布統計與逐學員具名明細並匯出 CSV；對某學員執行重置重考次數（限滿足條件者）與移除學員，動作後該學員狀態正確更新。

**Acceptance Scenarios**:

### 查詢與顯示

1. **Given** 教師於 ET03 進入，**When** 頂部下拉選單選擇某課程，**Then** 系統載入該課程之已加入學員清單
2. **Given** 已加入學員清單載入，**When** 系統顯示，**Then** 欄位包含：學員姓名、加入日期、完課狀態、學習進度、平均成績、最後活動時間
3. **Given** 學員加入但未開始學習，**When** 顯示，**Then** 完課狀態 = 「未開始」、學習進度 = 0%、平均成績欄位顯示「—」
4. **Given** 學員學習中，未完課，**When** 系統計算，**Then** 完課狀態 = 「進行中」；平均成績 = 該學員**已作答測驗**之最高分平均（未作答測驗排除於計算）
5. **Given** 學員完課（所有章節含測驗皆通過），**When** 系統計算，**Then** 完課狀態 = 「已完成」；平均成績 = 所有測驗之最高分平均
6. **Given** 學員已被移除，**When** 系統顯示，**Then** **不出現於「已加入」清單**；學習歷史紀錄完整保留於 DB

### 區塊 2 — 每次作答明細檢視（2026-07-02 新增）

7. **Given** 教師檢視區塊 2，**When** 系統載入，**Then** **一次列出所有曾作答之學員**（可展開清單，不設學員篩選）；展開某學員即見其於本課程各測驗之**歷次（每一次）attempt 清單**，每列顯示：作答時間、總分、是否及格
8. **Given** 教師點入歷次清單中之單次 attempt，**When** 系統載入，**Then** 顯示該次之**逐題明細**：題目、學員作答、正確答案、對錯、該題得分（依該 attempt 之題目 / 選項順序快照渲染）
9. **Given** 某學員於某測驗曾重考多次，**When** 教師檢視，**Then** 每次 attempt 皆可個別點入檢視（不限最近一次）；清單同時可見各次總分供對照
10. **Given** 某學員於某測驗尚無任何 attempt，**When** 教師檢視，**Then** 該測驗顯示「尚未作答」，無可展開之明細

### 區塊 3 — 問卷結果（2026-07-02 新增）

11. **Given** 所選課程**設有課後問卷**，**When** 教師進入「已加入」頁籤，**Then** 顯示「問卷結果」區塊；課程**無問卷時本區塊隱藏**
12. **Given** 教師於「問卷結果」區塊之**統計檢視**，**When** 系統載入，**Then** 顯示各題之**選項分布**（各選項人數與百分比）及**已填 / 未填人數**
13. **Given** 教師於「問卷結果」區塊之**明細檢視**，**When** 系統載入，**Then** 顯示**逐學員（具名）**之各題選擇
14. **Given** 教師於「問卷結果」區塊點「匯出 CSV」，**When** 系統處理，**Then** 依當前檢視產生 CSV 檔案供下載
15. **Given** 問卷尚無任何學員填答，**When** 教師檢視「問卷結果」區塊，**Then** 統計檢視顯示各選項 0 人、已填 0 / 未填 N；明細檢視顯示空清單提示

### 重置重考次數（位於區塊 2 作答明細之各測驗標題列，2026-07-02 移入）

16. **Given** 某學員於某測驗之已用重考次數 < 上限（或已及格 / 未作答），**When** 教師於區塊 2 該測驗標題列查看，**Then** 「重置重考次數」按鈕**不可點擊（停用）**
17. **Given** 某學員於某測驗之已用重考次數 = 上限**且尚未及格**，**When** 教師於區塊 2 該測驗標題列查看，**Then** 「重置重考次數」按鈕**可點擊**
18. **Given** 教師點「重置重考次數」並確認，**When** 系統處理，**Then** 該學員於該測驗之「已用重考次數」歸 0，恢復至滿配額狀態
19. **Given** 教師欲重置之測驗為**整個課程**之多測驗，**When** 系統處理，**Then** **僅以特定測驗為單位**（非整個課程一次重置）；每測驗需個別重置

### 移除學員

20. **Given** 該學員無作答中 attempt，**When** 教師點「移除學員」並確認，**Then** 立即標記 ET_ENROLLMENT 移除（REMOVED_AT 寫入）；學員 next request 被導至「您已被該課程移除」訊息頁
21. **Given** 該學員作答中（有 ET_QUIZ_ATTEMPT_M 狀態 = IN_PROGRESS），**When** 教師點「移除學員」，**Then** 系統警告「該學員作答中，移除後其當前 attempt 將保留並計入歷史。確定移除？」；確認後立即標記移除；學員當前 attempt 沿用快照完成
22. **Given** 學員被移除後，**When** 系統計算完課率，**Then** 該學員不計入分母（per 完課率定義「分母不含已移除學員」）

### 匯出

23. **Given** 教師於清單畫面點「匯出 CSV」，**When** 系統處理，**Then** 依當前篩選條件產生 CSV 檔案供下載；內容包含完整欄位

### 課程已關閉時之檢視（2026-07-02 新增）

24. **Given** 所選課程**已關閉**，**When** 教師進入 ET03「已加入」頁籤，**Then** 三區塊（學員清單、作答明細、問卷結果）仍可**閱覽**（含匯出 CSV）
25. **Given** 所選課程已關閉，**When** 教師檢視個別學員，**Then** 「重置重考次數」「移除學員」按鈕**禁用**；亦不可對該課程執行邀請（再開課後恢復）

## 系統訊息

各訊息類型（錯誤 / 警告 / 確認 / 成功 / 提示）定義見 [spec.md](spec.md) §Requirements。ET03 為多 US 共用畫面，本 US（學員學習狀況追蹤）使用流水號區段 001–099。

| 訊息代碼 | 類型 | 訊息內容 | 觸發情境 |
|---------|------|---------|---------|
| ET-MSG-ET03-001 | 確認 | 確定重置該學員此測驗之重考次數？ | 場景 18：重置重考次數 |
| ET-MSG-ET03-002 | 成功 | 已重置重考次數 | 場景 18：重置完成 |
| ET-MSG-ET03-003 | 警告 | 該學員作答中，移除後其當前 attempt 將保留並計入歷史。確定移除？ | 場景 21：移除作答中學員 |
| ET-MSG-ET03-004 | 成功 | 已移除學員 | 場景 20：移除學員成功 |
| ET-MSG-ET03-005 | 提示 | 尚未作答 | 場景 10：該學員於該測驗無 attempt |
| ET-MSG-ET03-006 | 提示 | 尚無學員填答問卷 | 場景 15：問卷結果區塊無填答 |
| ET-MSG-ET03-007 | 成功 | CSV 已匯出 | 場景 14/23：匯出 CSV |

## 相關 Clarifications 摘錄

- 重置重考次數限「次數歸 0 時」才可執行；故不存在「作答中重置」之競態
- 移除學員後 ET_ENROLLMENT 保留標記移除；學習歷史紀錄完整保留供稽核
- 每次作答明細依各 attempt 之 ET_QUIZ_ATTEMPT_M 順序快照渲染；學員端同等資料於 [spec_us6.md](spec_us6.md) US6 回看（2026-07-02 新增）
- 問卷填答**具名**；「問卷結果」統計與明細僅本課程教師（owner）與管理者可見；課程無問卷時該區塊隱藏（2026-07-02 新增；問卷結果為「已加入」頁籤之區塊 3，非獨立 tab）
- 加入來源顯示值：「**標籤帶入**」（原「模組預設」，2026-07-02 更名）/「Email 邀請」/「邀請碼」
- 課程已關閉（原「已停課」語意，2026-07-02 改為「已關閉」且可逆）時本頁為唯讀閱覽；再開課後管理動作恢復
