# User Story 7 — UCET001 課程列表瀏覽（ET01）

> 返回總檔：[spec.md](spec.md) | 模組：教育訓練文件管理（ET）

教師登入後預設首頁為 ET01 課程列表。本頁集中呈現所有教師建立之課程，亦為新增與編輯之入口。教師於頁面上方分頁切換「我建立的 / 全部課程」；下方搜尋區提供關鍵字（課程名稱）與關聯模組篩選，「全部課程」分頁額外提供「建立者」篩選。課程依關聯模組分組呈現卡片（採血 / 成分 / 檢驗 / 供應 / 醫務 / 報表與標籤 / 其他）；卡片含課程名稱、關聯模組 badge、章節數、發布狀態（草稿 / 已發布 / 停課中 / 已停課）、學員人數、建立者姓名。他人建立之課程可進入瀏覽（唯讀），無法編輯。

**Priority**: P2

**Why this priority**: 教師日常作業入口，但因本質為「列表瀏覽」屬輔助查詢功能；核心 CRUD 由 [spec_us3.md](spec_us3.md) US3 提供。

**Independent Test**: 多名教師建立各自之課程後，於 ET01「我建立的」分頁僅見自己建立者；切「全部課程」可見全部；點他人卡片進入後為唯讀模式。

**Acceptance Scenarios**:

1. **Given** 教師登入，**When** 系統依角色導向，**Then** 進入 ET01 課程列表（教師預設首頁）
2. **Given** 教師於 ET01 預設分頁為「我建立的」，**When** 系統載入，**Then** 僅顯示當前使用者建立之課程
3. **Given** 教師切至「全部課程」分頁，**When** 系統載入，**Then** 顯示所有教師建立之課程；搜尋區額外出現「建立者」篩選欄位
4. **Given** 教師於搜尋區輸入關鍵字「採血」，**When** 點查詢，**Then** 系統依課程名稱模糊比對列出結果
5. **Given** 教師選取關聯模組「成分」，**When** 點查詢，**Then** 系統列出該模組之課程
6. **Given** 系統載入清單，**When** 課程依關聯模組分組，**Then** 各組以課程卡片網格呈現，卡片含課程名稱、關聯模組 badge、章節數、發布狀態、學員人數、建立者姓名
7. **Given** 課程為當前使用者建立，**When** 卡片載入，**Then** 卡片右上**不顯示**「檢視」標籤；點擊進入 ET02 編輯模式
8. **Given** 課程為他人建立，**When** 卡片載入，**Then** 卡片右上顯示「檢視」標籤；點擊進入唯讀模式（所有編輯按鈕隱藏）
9. **Given** 課程狀態為「已停課」或「停課中」，**When** 卡片載入，**Then** 卡片標示對應狀態（pill / badge）；點擊進入後同樣為唯讀模式
10. **Given** 教師於搜尋區右側點「新增課程」按鈕，**When** 系統載入，**Then** 進入 ET02 新增模式（屬 [spec_us3.md](spec_us3.md) US3）

---

## 相關 Clarifications 摘錄

- 課程擁有者於建立當下記錄；本擁有者欄位永久不可變更
- 他人課程開啟時，畫面以唯讀模式呈現，所有編輯與其他操作按鈕皆不顯示

## 前置依賴

- 教師已於 [spec_us1.md](spec_us1.md) US1 被授予「教師」角色
- 至少有一門課程已建立（由 [spec_us3.md](spec_us3.md) US3）
