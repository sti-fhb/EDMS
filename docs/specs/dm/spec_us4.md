# User Story 4 — 文件詳細頁瀏覽（UCDM04 / DM02）

> 返回總檔：[spec.md](spec.md) | 模組：文件管理（DM）

## User Story

使用者於文件詳細頁閱讀目前發布版本、下載原檔或線上預覽（PDF 與圖片可預覽，Office 檔僅下載），並可展開版本歷程檢視所有版本。具編輯者角色者另可由此進入編輯新版本或發起廢止申請。上方標題列僅顯示識別與狀態資訊（文件名稱 / DOC_ID / 目前版本 / 狀態），描述性 metadata 統一由右側「文件資訊」面板呈現。

**Priority**: P1 — 詳細頁是閱讀文件內容、取得目前版本、檢視版本歷程之核心畫面，亦為編輯 / 廢止之入口。

**Independent Test**: 使用者由 DM01 點入詳細頁，可下載目前版本 PDF 並線上預覽；展開版本歷程見各版號 / 撰寫者 / 發布時間 / 核准者 / 變更摘要；舊版本僅可預覽不可下載；具編輯者角色者見「編輯新版本」與「廢止此文件」入口。

### Acceptance Scenarios

1. **Given** 使用者由 DM01 或 DM00 點選文件，**When** 進入詳細頁，**Then** 上方標題列顯示文件名稱 / DOC_ID / 目前版本 / 狀態；右側「文件資訊」顯示分類 / 作者 / 發布日期 / 核准者 / 核准時間 / 標籤（不於上方重複）
2. **Given** 目前發布版本為 PDF 或圖片，**When** 顯示文件檔案區，**Then** 提供線上預覽與下載
3. **Given** 目前發布版本為 Office 檔（Word / Excel / PPT），**When** 顯示文件檔案區，**Then** 顯示「下載原檔以本機應用程式開啟」、不提供線上預覽
4. **Given** 使用者按「版本歷程」，**When** 抽屜展開，**Then** 以時間軸列出所有版本（含已被取代之歷史版本）之版號 / 撰寫者 / 發布時間 / 核准者 / 變更摘要
5. **Given** 版本歷程中之舊版本，**When** 使用者操作，**Then** 僅提供預覽、**不開放下載**（避免誤用過時規範）；目前發布版本可下載
6. **Given** 使用者具編輯者角色，**When** 檢視詳細頁，**Then** 額外顯示「編輯新版本」與「廢止此文件」入口（→ [spec_us5.md](spec_us5.md) / [spec_us8.md](spec_us8.md)）
7. **Given** 文件處於新版本送審中或廢止待簽核狀態，**When** 編輯者檢視，**Then** 編輯與廢止入口暫時失效（避免兩個送審週期並存）
8. **Given** 使用者自 [spec_us10.md](spec_us10.md)（已廢止文件查詢）進入，**When** 詳細頁渲染，**Then** 以 read-only 模式呈現：隱藏「文件檔案 + 文件資訊」整段、自動展開版本歷程、所有版本僅可預覽

## Functional Requirements

- **FR-001**: 詳細頁上方標題列 MUST 僅顯示識別與狀態資訊（文件名稱、DOC_ID、目前版本、狀態：已發布 / 已廢止）；描述性 metadata（分類 / 作者 / 發布日期 / 核准者 / 核准時間 / 標籤）MUST 統一由右側「文件資訊」面板呈現，不於上方重複
- **FR-002**: 文件檔案區 MUST 依格式提供存取：**PDF 與圖片**內嵌線上預覽 + 下載；**Office 檔**僅下載（顯示「下載原檔以本機應用程式開啟」、不預覽）
- **FR-003**: 系統 MUST 提供「版本歷程」抽屜，列出該文件所有版本（含歷史版本）之版號 / 變更摘要 / 撰寫者 / 核准者 / 發布時間
- **FR-004**: **僅目前發布版本可下載**（不分格式）；已被取代之舊版本 MUST 僅提供預覽、不開放下載；稽核需取得舊版原始檔須聯絡 DM_ADMIN
- **FR-007**: 使用者**下載目前發布版**時系統 MUST 寫入一筆閱讀紀錄 `DM_DOC_READ`（使用者 × 文件 × 版本 × 時間），作為閱讀 KPI（[spec_us13.md](spec_us13.md)）之「已看」判定；**預覽不記錄**；同一使用者對同一版本重複下載以人去重
- **FR-005**: 系統 MUST 依當前使用者角色顯示動作入口：具編輯者角色顯示「編輯新版本」與「廢止此文件」；文件處於送審中 / 廢止待簽核時該入口 MUST 暫時失效
- **FR-006**: 自已廢止文件查詢進入時系統 MUST 以 read-only 模式呈現：隱藏「文件檔案 + 文件資訊」整段、自動展開版本歷程、所有版本僅可預覽不可下載、上方紅色 banner 標示廢止資訊（廢止時間 / 申請人 / 核准者 / 廢止原因 / **廢止附件下載，如有**）
