# User Story 3 — 文件庫與檢索（UCDM03 / DM01）

> 返回總檔：[spec.md](spec.md) | 模組：文件管理（DM）

## User Story

使用者於文件庫以多條件搜尋已發布文件（關鍵字 / 分類 / 作者 / 標籤多選 AND / 發布日期區間），並可進入文件詳細頁。當分類選「系統操作手冊」時，額外顯示「關聯作業項目（func_name）」下拉，依作業項目檢索對應手冊（達成線上操作手冊能力）。具編輯者角色者另可由清單進入新增文件。

**Priority**: P1 — 文件庫是閱讀者最主要入口、也是線上操作手冊檢索與編輯者新增之起點；為 DM 核心讀取作業。

**Independent Test**: 使用者登入後於 DM01 以關鍵字 + 標籤搜尋，得到僅含「已發布」目前版本之清單；選分類「系統操作手冊」後出現 func_name 下拉，選一作業項目可列出對應手冊；具編輯者角色者見「新增文件」入口。

### Acceptance Scenarios

1. **Given** 使用者已登入，**When** 進入 DM01 並以關鍵字（文件名 / 變更摘要）、分類、作者、標籤（多選 AND）、發布日期區間搜尋，**Then** 系統列出符合條件之文件，依發布時間由新到舊分頁顯示
2. **Given** 使用者未輸入任何條件即查詢，**When** 送出，**Then** 系統列出全部「已發布」目前版本文件
3. **Given** 搜尋結果清單，**When** 顯示，**Then** 欄位含文件名稱 / 分類 / 發布日期 / 作者 / 標籤（標籤以灰色文字頓號分隔）；系統操作手冊類額外顯示對應 func_name
4. **Given** 文件處於送審中 / 草稿 / 已退回 / 已廢止或為歷史版本，**When** 使用者搜尋，**Then** 該等文件**不出現**於清單；廢止待簽核期間文件因仍對外有效**持續顯示**
5. **Given** 使用者選分類「系統操作手冊」，**When** 條件區渲染，**Then** 額外顯示「關聯作業項目（func_name）」下拉（選項來自受控清單，不可自由輸入）
6. **Given** 使用者於「系統操作手冊」下拉選定某 func_name，**When** 搜尋，**Then** 系統列出對應之系統操作手冊（因同一 func_name 至多一份已發布手冊，結果唯一）
7. **Given** 使用者點任一文件列，**When** 點擊，**Then** 進入 UCDM04 文件詳細頁
8. **Given** 使用者具編輯者角色，**When** 檢視清單，**Then** 右側顯示「新增文件」入口（→ [spec_us5.md](spec_us5.md) 新增模式）；無編輯者角色則不顯示

## Functional Requirements

- **FR-001**: 系統 MUST 提供多條件搜尋：關鍵字（比對文件名稱、變更摘要 / 首版摘要，簡易模糊、不含全文檢索）、分類、作者（撰寫者姓名）、標籤（受控庫多選、多標籤 **AND**）、發布日期區間
- **FR-002**: 清單 MUST 僅呈現「已發布」狀態之目前發布版本，且 MUST 包含「廢止待簽核」文件（仍對外有效）；MUST NOT 顯示送審中 / 草稿 / 已退回 / 已廢止文件與歷史版本
- **FR-003**: 清單欄位 MUST 含文件名稱 / 分類 / 發布日期 / 作者 / 標籤；標籤以**灰色文字頓號分隔**顯示（不用彩色 pill）；系統操作手冊類額外顯示對應 func_name
- **FR-004**: 當分類條件為「系統操作手冊」時系統 MUST 額外顯示「關聯作業項目（func_name）」下拉（選項來自受控清單），供依作業項目檢索；其他分類不顯示此條件
- **FR-005**: 系統 MUST 依排序規則（發布時間 DESC）分頁呈現；無符合結果時顯示空狀態提示（DM-MSG-DM01-001）
- **FR-006**: 系統 MUST 依當前使用者角色決定「新增文件」入口之顯示（僅編輯者可見）；點擊進入 [spec_us5.md](spec_us5.md) 新增模式

> **2026-06-26 變更**：原 FR-007「主系統各畫面『線上系統操作手冊』按鈕依 func_name 跨系統反查 DM」已**移除**；DM 與主系統間不再有此介接。「線上操作手冊」能力改由 DM 內部檢索（FR-004）達成。
