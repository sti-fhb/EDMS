# User Story 5 — 文件新增與編輯（UCDM06 / DM03）

> 返回總檔：[spec.md](spec.md) | 模組：文件管理（DM）

## User Story

編輯者新增文件（建立 DOC_ID、首版）或對既有文件上傳新版本，填寫必填欄位、選用受控標籤、指定審核者後送交簽核；亦可存為草稿續編。編輯既有文件時文件名稱 / 分類 / 關聯作業項目為唯讀（身份識別屬性）。系統操作手冊類須單選 func_name 且同一 func_name 至多對應一份已發布手冊。

**Priority**: P1 — 文件產製為 DM 之核心輸入作業；無新增 / 編輯則無文件可簽核、發布、閱讀。

**Independent Test**: 編輯者於 DM03 新增模式填妥名稱 / 分類 / 摘要 / 標籤、指定首版版號、上傳 PDF、選審核者後送簽，文件進入「送審中」；於 DM02 對既有文件「編輯新版本」上傳新檔送簽；上傳 Office 檔時跳預覽提醒；存草稿後可於 [spec_us9.md](spec_us9.md) 個人專區續編。

### Acceptance Scenarios

1. **Given** 編輯者由 DM01 點「新增文件」，**When** 進入新增模式，**Then** 可填文件名稱 / 分類 / 標籤（多選、選填）、首版版號（系統建議 v1.0、可調）、首版摘要，並上傳檔案
2. **Given** 分類選「系統操作手冊」，**When** 表單渲染，**Then** 額外要求單選「關聯作業項目（func_name）」
3. **Given** 必填欄位（文件名稱 / 分類 / 變更摘要 / 指定審核者）未填，**When** 送簽，**Then** 系統阻擋並提示（DM-MSG-DM03-001）
4. **Given** 編輯者由 DM02 點「編輯新版本」，**When** 進入編輯模式，**Then** 文件名稱 / 分類 / 關聯作業項目為**唯讀**，僅可改檔案 / 變更摘要 / 標籤 / 指定審核者 / 版本號（系統建議 +0.1 / Major / Minor、可調）
5. **Given** 編輯者上傳非可預覽之 Office 檔（Word / Excel / PPT），**When** 上傳完成，**Then** 系統跳提醒「閱讀者於詳細頁無法線上預覽，僅能下載」，可改傳 PDF 或選「仍使用此檔案」續行（DM-MSG-DM03-002）；上傳 PDF 或圖片不觸發提醒
6. **Given** 編輯者選擇指定審核者，**When** 下拉渲染，**Then** 名單為具審核者角色之使用者且**排除自己**（職責分離）
7. **Given** 系統操作手冊指定之 func_name 已被其他「已發布」手冊使用，**When** 送簽 / 發布，**Then** 系統阻擋並提示「此關聯作業項目已有對應手冊」（DM-MSG-DM03-003）
8. **Given** 編輯者點「儲存為草稿」，**When** 儲存，**Then** 文件存為草稿、出現於 [spec_us9.md](spec_us9.md) 個人專區草稿區供續編；DOC_ID 於草稿建立時即配號
9. **Given** 文件處於廢止待簽核狀態，**When** 編輯者嘗試上傳新版本，**Then** 系統阻擋並提示（不可兩個送審週期並存）（DM-MSG-DM03-004）
10. **Given** 表單有未儲存變更，**When** 點「取消新增 / 取消編輯」，**Then** 跳確認對話框（DM-MSG-DM03-005）

## Functional Requirements

- **FR-001**: 新增模式 MUST 建立新文件 DOC_ID（格式 `DM-{分類碼}-{6 位流水號}`，流水號依分類獨立、草稿建立時配號）；必填欄位：文件名稱、分類、變更摘要 / 首版摘要、指定審核者（送審時）；標籤為選填多選（受控庫挑選、不可自由輸入）
- **FR-002**: 分類為「系統操作手冊」時 MUST 額外要求單選 func_name；且 MUST 於送簽 / 發布檢核「同一 func_name 至多對應一份已發布手冊」，重複則阻擋
- **FR-003**: 編輯模式（上傳新版本）MUST 將文件名稱 / 分類 / 關聯作業項目設為唯讀；僅可修改檔案、變更摘要、標籤、指定審核者、版本號。如需改名稱 / 分類，須廢止原文件後另行新增
- **FR-004**: 版本號 MUST 由系統自動建議（+0.1 / Major / Minor），撰寫者可採建議值或自行修改（可直接編輯，如 v2.0-RC1）；變更摘要欄於首版顯示「首版摘要」、新版本顯示「變更摘要」（同欄 CHANGE_SUMMARY）
- **FR-005**: 上傳 MUST 支援 PDF / Word / Excel / PowerPoint / 圖片，**每版本單一檔案**，單檔上限預設 50 MB（由 DM_PARAM 控制）；上傳**不支援預覽之 Office 格式**時 MUST 跳提醒並於檔案下方持續顯示橘色警示條（PDF / 圖片不觸發）
- **FR-006**: 指定審核者下拉 MUST 僅列具審核者角色之使用者且**排除撰寫者本人**（不可自審自核）
- **FR-007**: 系統 MUST 支援「儲存為草稿」（不送簽、可續編，列於個人專區草稿區）；送簽後文件轉「送審中」並通知指定審核者（→ [spec_us6.md](spec_us6.md)）
- **FR-008**: 文件處於廢止待簽核狀態時系統 MUST 阻擋上傳新版本（避免兩個送審週期並存）；表單有未儲存變更時取消 MUST 二次確認

## 系統訊息

| 訊息代碼 | 類型 | 訊息內容 | 觸發 / 對應 FR |
|---------|------|---------|---------------|
| DM-MSG-DM03-001 | 錯誤 | 請填寫文件名稱、分類、變更摘要並指定審核者 | FR-001 必填未填 |
| DM-MSG-DM03-002 | 警告 | 此檔案為 Office 格式，閱讀者於詳細頁將無法線上預覽，僅能下載原檔 | FR-005 非預覽格式 |
| DM-MSG-DM03-003 | 錯誤 | 此關聯作業項目已有對應之已發布手冊，請改選其他作業項目 | FR-002 func_name 重複 |
| DM-MSG-DM03-004 | 錯誤 | 此文件正進行廢止簽核，無法同時上傳新版本 | FR-008 送審週期衝突 |
| DM-MSG-DM03-005 | 確認 | 表單有未儲存之變更，確定離開？ | FR-008 取消確認 |
| DM-MSG-DM03-006 | 成功 | 已送交簽核，已通知指定審核者 | FR-007 送簽完成 |
| DM-MSG-DM03-007 | 成功 | 已儲存為草稿 | FR-007 存草稿 |
