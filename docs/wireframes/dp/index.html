💡 內部設計註記（UCDP005 使用者管理）
  • 帳號為 DP_USER，ET / DM 共用；停用後兩系統同步失效。
  • 帳號權限以最小權限為原則；唯一識別、不得共用帳號。
  • 連續登入失敗達上限（參數，預設 5）自動鎖定 → 此頁可手動解鎖；亦可逾時自動解鎖。
  • 連續 90 日未登入之帳號由系統自動停用 / 鎖定。
  • 使用者可自助註冊即用；管理者代建之帳號亦直接可用（不寄開通確認信）。
  • 此頁不設角色指派——ET / DM 之業務角色（教師/學員、DM 各角色）由各模組自己的權限管理頁（如 ET07）維護（權限完全分開做）；平台不碰業務角色、不做全域選單 RBAC。
  • DP 自身功能（使用者管理 / 系統參數 / 操作記錄）以 DP_USER.IS_PLATFORM_ADMIN 旗標把關（平台管理員，非業務角色體系）；平台管理員看不到 ET / DM 業務參數與範本。
  • 所有帳號異動寫入操作記錄（DP_AUDIT_LOG）。

💡 內部設計註記（UCDP006 系統參數維護）
  • 參數表 DP_PARAM_M / DP_PARAM_D；異動記錄歷程（稽核）。
  • 密碼長度分級（一般 8 / 特權 12）保留以對齊國防部資安法規。
