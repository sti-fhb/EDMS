關閉課程
關閉後此課程對學員轉為唯讀回看模式；可隨時以「再開課」恢復開放（需重設起訖時間，非不可逆）。
  • ET01 課程列表此課程將顯示「已關閉」狀態 pill；教師可進入檢視並執行「再開課」
  • 學員端 ET04 / ET05 課程仍可見、可唯讀回看已學過的內容；不能再累積進度、作答測驗、解鎖新章節、填寫問卷
  • 學員嘗試以邀請碼加入時顯示「此課程目前關閉中」（邀請碼於關閉期間失效，再開課後恢復）
  • 關閉當下若有學員正在作答測驗：允許該次作答完成並計分；之後禁止開始新的作答
  • ET03 學員教師端可繼續查看歷史完課狀態 / 成績 / 明細，但不可再邀請新學員、不可重置重考次數
  • 關閉期間停止每週統計與提醒排程；再開課後恢復
另有「到期自動關閉」：課程到達訖止時間時，系統自動將狀態轉為已關閉（效果與手動關閉相同）。

  • 邀請碼恢復有效；每週統計與提醒排程恢復執行
  • 截止前加急提醒依新訖止時間重新計算（每課一次重新起算）
  • 關閉 / 再開課可重複多次
