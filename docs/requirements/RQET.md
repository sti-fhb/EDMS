## 模組概述

教育訓練模組（ET）提供血液中心內部人員的線上教育訓練功能，含課程建立與編輯、教材管理、線上測驗、學員學習狀況追蹤、邀請與權限管理。ET 與 DM 共用 user table（同帳號可登入兩系統）；教材文件統一由 DM 模組提供。

### 操作角色

| 角色 | 職責範圍 |
|------|---------|
| 管理者 | 權限管理 — 設定其他帳號於 ET 的角色 |
| 教師 | 課程安排 — 建立 / 編輯 / 停課課程、章節、教材、測驗；邀請學員；查看學員學習狀況 |
| 學員 | 加入課程、觀看教材、進行線上測驗、查看個人學習進度 |
| eMail Server | 外部郵件系統，負責寄發課程邀請信 |

> 角色可重複指派；新使用者首次登入自動授予「學員」角色。

### 作業流程

```
教師作業
────────
1. 建立新課程（名稱、關聯模組、描述）
2. 編排章節、上傳教材（影片 / DM 文件 / 說明文字）
3. 建立線上測驗（題庫、單選/多選、自動閱卷）
4. 發布課程，邀請學員（Email 邀請 / 邀請碼 / 模組預設帶入）
5. 於 ET03 學員頁追蹤完課狀態、學習進度、平均成績
6. 課程結束後可選擇「停課」（不可逆）

學員作業
────────
1. 從 Email 邀請或邀請碼加入課程
2. 於 ET04 我的課程選擇課程進入學習（ET05）
3. 依序學習章節（影片、文件、說明文字）
4. 完成章節後進行測驗（ET06），系統自動閱卷
5. 查看個人學習進度與測驗成績

管理者作業
──────────
1. 於 ET07 設定其他帳號之角色（管理者 / 教師 / 學員）
2. 維護使用者×業務模組對應，用於課程預設帶入
```
