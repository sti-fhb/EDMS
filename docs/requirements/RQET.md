## 模組概述

教育訓練模組（ET）提供血液中心內部人員的線上教育訓練功能，含課程建立與編輯、教材管理、線上測驗、課後回饋問卷、學員學習狀況追蹤、排程統計提醒、邀請與權限管理。ET 與 DM 共用 `DP_USER`（同帳號可登入兩系統）；教材文件統一由 DM 模組提供。

### 操作角色

| 角色 | 職責範圍 |
|------|---------|
| 管理者 | 權限管理 — 設定其他帳號於 ET 的角色；維護受訓單位標籤庫與使用者×標籤對應；維護通知信範本與排程參數 |
| 教師 | 課程安排 — 建立 / 編輯 / 關閉 / 再開課程、章節、教材、測驗、課後問卷；設定課程起訖時間；邀請學員；查看學員學習狀況、每次作答明細與問卷結果 |
| 學員 | 加入課程、觀看教材、進行線上測驗、填寫課後問卷、查看個人學習進度 |
| 系統自動 | 發布時標籤自動邀請＋寄信、到期自動關閉、每週統計與提醒排程、截止前加急提醒 |
| eMail Server | 外部郵件系統，負責寄發課程邀請、提醒、週報等信件 |

> 角色可重複指派；新使用者首次登入自動授予「學員」角色。

### 作業流程

```
教師作業
────────
1. 建立新課程（名稱、受訓單位標籤（多選）、起訖時間、描述）
2. 編排章節、上傳教材（影片 / DM 文件 / 說明文字）
3. 建立線上測驗（題庫、單選/多選、自動閱卷）；建立課後回饋問卷（0~1 份）
4. 發布課程 → 系統自動邀請標籤對應人員並寄通知信；另可 Email 邀請 / 邀請碼補件
5. 於 ET03 學員頁追蹤完課狀態、學習進度、成績、每次作答明細、問卷結果
6. 到訖止時間系統自動關閉（或教師手動關閉）；關閉後可「再開課」（重設起訖時間）

學員作業
────────
1. 由標籤自動加入（收到通知信），或以 Email 邀請 / 邀請碼加入課程
2. 於 ET04 我的課程選擇課程進入學習（ET05；起始時間到才看得到課程）
3. 依序學習章節（影片、文件、說明文字）
4. 完成章節後進行測驗（ET06），系統自動閱卷；可回看每次作答明細
5. 完課後填寫課後問卷；查看個人學習進度與測驗成績

管理者作業
──────────
1. 於 ET07 設定其他帳號之角色（管理者 / 教師 / 學員）
2. 維護受訓單位標籤庫（新增 / 修改 / 停用 / 啟用）與使用者×標籤對應（用於課程自動邀請）
3. 維護通知信範本與排程參數

系統排程
────────
1. 每週一 10:00 統計各開放中課程看課狀況 → 週報寄教師與管理者；進度 0% 學員寄未看提醒（一人一信彙整）
2. 每日檢查課程訖止前 3 天 → 對未完課學員寄加急提醒（每課僅一次）
```
